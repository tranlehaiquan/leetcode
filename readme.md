# Algorithm and data structure

This repository contains implementations of various algorithms and data structures, along with solutions to LeetCode problems.

## Getting Started

To set up the project and install dependencies, use [Bun](https://bun.sh/):

```bash
bun install
```

## Running Tests

To run all tests:

```bash
bun test
```

To run tests with coverage:

```bash
bun test --coverage
```

## Folder Structure

- `data-structure`: Implementations of various data structures.
- `problems`: Solutions to LeetCode problems.
- `leetcode`: Solutions to Leetcode problems (another category)
- `binarySearch`: Binary Search algorithm.
- `breadthFirstSearch`: Breadth-First Search algorithm.
- `dijkstra`: Dijkstra's algorithm.
- `quickSort`: Quick Sort algorithm.
- `selectionSort`: Selection Sort algorithm.
- `set`: Set data structure implementations.
- `utils`: Utility functions.
- `interview-tests`: Solutions for interview tests.
